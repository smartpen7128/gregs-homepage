# Greg's Homepage - Bookmark Manager

A simple bookmark manager built with HTML, CSS, JavaScript, and Fluent UI.

## Features

- Create adjustable columns
- Add cards with titles and links
- Drag and drop cards between columns and reorder within columns
- Edit titles and URLs
- Data persists in localStorage

## How to Use

1. Open `index.html` in your web browser.
2. Click "Add Column" to create a new column.
3. Click "Add Card" in a column to add a bookmark.
4. Drag cards to reorder them.
5. Click on titles to edit them.
6. Double-click on cards to edit URLs.
7. Use the X button to delete cards.
8. Resize columns by dragging the edges.

No server required - runs entirely in the browser.
